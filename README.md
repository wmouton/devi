# Devi

**Devi** is a command-line tool for managing API tokens with expiration dates.

## Quickstart

1. **Install Devi:**

   ```sh
   cargo install devi
   ```

2. **Run Devi:**

   For a list of available commands:

   ```sh
   devi help
   ```
